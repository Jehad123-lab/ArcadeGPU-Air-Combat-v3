# Changelog: ArcadeGPU Flight Simulator

## [1.1.0] - 2026-05-08
### Added
- **Flight Simulator**: Full implementation of the flight simulator mechanics.
- **Flight Physics**: Realistic roll, pitch, and yaw implemented usingJolt Physics integration.
- **Procedural Plane**: Automated generation of plane meshes.

### Changed
- **Rebrand**: Project transitioned from Tank Pursuit to Flight Simulator.
- **Removed**: Tank combat assets and game logic.

### Fixed
- **Camera Sensitivity**: Smoothed camera controls for flight.

## [1.0.0] - Initial Release
- Basic flight mechanics.
- Jolt Physics integration for flight.
- Procedural environment generation.
